## 模板使用说明

1. **每条反馈一份独立文件**，建议命名：`Phase14-体验反馈-NNN-{简短主题}.md`，存放于 `docs/launch/feedback/`。
2. 体验方仅填写 P0~P3 优先级，PM 决定是否阻塞发布；P0 必须当场处理，P1 必须在签字前给出解决方案。
3. 鼓励附截图/录屏（保存到 `docs/launch/feedback/assets/` 并相对引用）。
4. 反馈汇总写入 `docs/launch/Phase14.7-三签字流程.md` §6 体验方反馈区。
