## 一、 品牌定义

### 1. 核心价值
*   **中文名：** **青萍**
*   **英文名：** **Ripple**
*   **定位：** 一款以“水文生态”为隐喻的下一代创意生产与协作系统。
*   **哲学：** **Small Input, Big Wave.**

### 2. 登录页 · 核心文案
> 我们厌倦了钢铁的冰冷，于是选择了水的形状。  
> 在这里，灵感如**云霓**般游离，似**雾气**般难以捕捉。  
>   
> 不必急于定型。  
> 你可以在**造浪池**中，任由思绪分流、合流，或是**蒸发**掉那些无用的噪音；  
> 直到某一刻，当你准备好释放，便从**决堤口**奔涌而出，  
> 让原本微小的**涟漪**，终成席卷现实的**巨浪**。  
>   
> 这，便是「青萍」。  
> 一个关于**生长、流动与爆发**的隐秘生态。  
>   
> **「 投石 · 以此开篇 」**

## 二、 系统架构：水文全链路闭环（重大升级）

我们将创意的生产过程解构为“气态 → 液态 → 固态”的自然形态演变。**在液态（造浪池）阶段，系统底层已从“线性流”进化为“拓扑网”。**

### 1. 气态 · 无形之思（灵感捕捉）
*   **模块命名：** **云霓 (Cloud)**
*   **功能定义：** 移动端/快捷入口，用于接收最原始、最碎片化的灵感（语音、涂鸦、截图）。
*   **状态描述：** 此时的灵感如云雾般飘忽不定，尚未成形，无法捉摸。
*   **交互动作：** **凝露**（将雾气凝结成水滴，使其具备进入下一阶段的条件）。

### 2. 液态 · 流动之力（核心创作）—— **【架构升级】**
*   **模块命名：** **造浪池 (Studio)**
*   **底层逻辑：** **图结构 (Graph Structure)**。
    *   每一个录入的灵感都是一个**“浮萍节点” (Node)**。
    *   AI 通过语义分析，自动计算节点间的**“亲缘关系”**（相似度、因果链、对立面），形成**“暗流连线” (Edge)**。
*   **核心机制（新增）：**
    *   **生根发芽 (Rooting & Branching)：** 激活一个节点后，AI 像根系一样向下挖掘（追问“为什么？”），或像藤蔓一样横向攀爬（联想“还有什么类似？”）。
    *   **织造网络 (Weaving)：** 用户手动拖拽两个节点（如“青萍”与“马具工程”），AI 在连线上生成**“中间态”**（Mid‑point），解释两者的隐喻关联。
    *   **分流/合流：** 保留原有的多方案试错能力，但在图结构中表现为节点的分叉与聚合。

### 3. 固态 · 现实之果（资产沉淀）
*   **模块命名：** **冰山 (Iceberg)**
*   **功能定义：** 最终交付与资产管理。
*   **升级点：** 资产不再是一条线性的历史记录，而是一张**“灵感三角洲”拓扑图**。用户可以清晰地看到，最终成品（冰山一角）是由哪些早期的“云霓”节点，经过怎样的“暗流”汇聚而成的。

## 三、 UI/UX 设计落地指南（图结构适配）

### 1. 造浪池视图模式切换
*   **俯瞰模式 (Delta View)：** 全局视角。节点像荷叶漂浮，连线是流动的曲线，线上有粒子特效表示 AI 正在推理。距离近的节点连线粗亮（强关联），距离远的连线淡细（弱关联）。
*   **聚焦模式 (Deep Dive)：** 选中某一节点，右侧属性栏变为**“生长引擎”**，显示该节点的“根系”和“藤蔓”建议。

### 2. 交互语言体系（按钮词典·增补）
| 常规功能 | **青萍命名** | 隐喻解析 |
| :--- | :--- | :--- |
| 新建关联 / Link | **引流** | 在两个节点间开辟水道，让思想流通。 |
| 查找路径 / Pathfind | **探源** | 寻找两个看似无关节点之间的最短逻辑链路。 |
| 断开连接 / Unlink | **截流** | 切断无效或错误的关联。 |

### 3. 空状态 (Empty State)
*   **禁用：** “无数据”、“404 Not Found”。  
*   **启用：** 展示一幅**平静的湖面**插画。  
*   **文案：** “**此处风平浪静，等待第一缕青萍。**”

### 4. 色彩心理学 (Color Scheme)
*   **主色调 — 青碧色 (#2E8B57)：** 代表“青萍”的生机、起点、自然。  
*   **辅助色 — 日出橙 (#FF7F50)：** 代表“燎原”的爆发力、行动点（CTA 按钮统一使用此色，形成冷暖对冲）。  
*   **背景色 — 深海蓝 (#003366)：** 代表深度、专业与沉淀（适用于后台管理或“冰山”模块）。

## 四、 交互语言体系（按钮词典）

### 1. 生成与创造 (Creation)
| 常规文案 | **青萍命名** | 隐喻解析 |
| :--- | :--- | :--- |
| 新建 / New | **投石** | 向平静水面投入石子，打破平静，这是一切创意的起点。 |
| 生成 / Generate | **降雨** | 灵感如雨水，自上而下地补给，让水位上涨。 |
| 导入 / Import | **引流** | 把外面的水（资源）引到自己的池塘里来。 |
| 随机 / Shuffle | **湍流** | 打破僵局，制造混乱，以此激发新的秩序。 |

### 2. 编辑与修正 (Editing)
| 常规文案 | **青萍命名** | 隐喻解析 |
| :--- | :--- | :--- |
| 保存 / Save | **固形** | 将流动的液体暂时凝固，便于存储。 |
| 复制 / Copy | **分蘖 (niè)** | 如禾本科植物分枝，一个好的创意可以生出多个分株。 |
| 粘贴 / Paste | **浸润** | 像水墨晕染一样，自然融合，不生硬。 |
| 撤销 / Undo | **退潮** | 如潮水退回，准备下一次冲击。 |
| 删除 / Delete | **蒸发** | 温和的告别，它并未消失，只是换了种形态。 |

### 3. 发布与分享 (Distribution)
| 常规文案 | **青萍命名** | 隐喻解析 |
| :--- | :--- | :--- |
| 发布 / Publish | **决堤口** | 积蓄已久的能量，势不可挡地奔涌而出。 |
| 导出 / Export | **入海** | 创意离开系统，融入更广阔的海洋（市场/世界）。 |
| 分享 / Share | **抛饵** | 抛出一个诱饵，期待引来更多的鱼（关注/互动）。 |
| 发送 / Send | **漂流瓶** | 带着信物的漂流，充满未知和浪漫。 |

### 4. AI 与智能辅助 (AI & Analysis)
| 常规文案 | **青萍命名** | 隐喻解析 |
| :--- | :--- | :--- |
| 搜索 / Search | **探源** | 寻找灵感的发源地。 |
| 分析 / Analyze | **透视** | 看透表象，看清水下（数据）的本质。 |
| 优化 / Optimize | **疏浚** | 疏通堵塞，让创意的流速更快。 |
| 总结 / Summary | **沉淀** | 去伪存真，留下最精华的部分（冰山一角）。 |
