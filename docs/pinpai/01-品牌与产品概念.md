## 第一篇 · 品牌（Why）

### 一、起源：当风遇见水

> "夫风生于地，起于青萍之末。" —— 宋玉《风赋》

在被算法和进度条包围的时代，创作者陷入一种悖论：**算力前所未有，从容前所未有地稀缺**。

我们见过太多这样的场景：一个绝妙的念头像流星划过夜空，转瞬即逝；一个宏大的项目因为中途的卡顿，最终沦为硬盘里的一堆废稿。

如果创意不是冰冷的代码，它应该是什么样子？答案藏在两千年前的一阵风里。**风生于大地，始于浮萍**。它没有形状，却能塑造山川；它看似柔弱，却能掀起巨浪。

于是，「青萍 (Ripple)」诞生了。

### 二、隐喻：水文生态

「青萍」不相信"工业风"。我们厌倦了齿轮的咬合声和红色的错误警告。**创意的本质是流体**。

| 模块 | 形态 | 隐喻 |
| :--- | :--- | :--- |
| **云霓 (Cloud)** | 气态 | 灵感的呼吸——飘忽、不索取归类、可任由蒸发 |
| **造浪池 (Studio)** | 液态 | 织造思维的湍流——网状蔓延，不再单线奔跑 |
| **冰山 (Iceberg)** | 固态 | 资产的沉淀——不仅交付作品，更交付创作基因 |

### 三、使命：微小，即伟大

中文里"青萍"常被误解为微不足道的浮萍。但在「青萍」的世界观里：**渺小，是伟大的序章。**

我们致敬每一位在平凡生活中寻找诗意的创作者。备忘录里记下的一句歌词、深夜画的一张潦草草图——在这里都不是垃圾，而是**火种**。

## 第二篇 · 产品（What）

### 一、核心定义

- **中文名：** 青萍
- **英文名：** Ripple
- **定位：** 一款以"水文生态"为隐喻的下一代创意生产与协作系统
- **哲学：** **Small Input, Big Wave.**

### 二、登录页文案（核心母本）

> 我们厌倦了钢铁的冰冷，于是选择了水的形状。
> 在这里，灵感如**云霓**般游离，似**雾气**般难以捕捉。
>
> 不必急于定型。
> 你可以在**造浪池**中，任由思绪分流、合流，或是**蒸发**掉那些无用的噪音；
> 直到某一刻，当你准备好释放，便从**决堤口**奔涌而出，
> 让原本微小的**涟漪**，终成席卷现实的**巨浪**。
>
> 这，便是「青萍」。
> 一个关于**生长、流动与爆发**的隐秘生态。
>
> **「 投石 · 以此开篇 」**

### 三、系统架构：水文全链路闭环

创意过程被解构为"气态 → 液态 → 固态"的自然形态演变。

#### 1. 云霓 (Cloud) · 气态

- **功能定义：** 移动端 / 快捷入口，接收最原始、最碎片化的灵感（语音、涂鸦、截图）
- **状态描述：** 灵感如云雾般飘忽不定，尚未成形
- **核心交互：** **凝露**（将雾气凝结成水滴，使其具备进入下一阶段的条件）
- **技术实现：** 见 [G2 云霓灵感采集模块](../system-design/G2-云霓-灵感采集模块设计.md)

#### 2. 造浪池 (Studio) · 液态

- **底层逻辑：** **图结构 (Graph Structure)**——每个灵感是浮萍节点，AI 通过语义分析自动计算"暗流连线"
- **核心机制：**
  - **生根发芽 (Rooting & Branching)**：AI 像根系向下追问"为什么"，或像藤蔓横向联想"还有什么"
  - **织造网络 (Weaving)**：用户拖拽两节点，AI 在连线上生成"中间态"
  - **分流 / 合流**：保留多方案试错能力，图结构中表现为节点的分叉与聚合
- **技术实现：** 见 [造浪池 Delta View 设计](../system-design/D1-造浪池-Delta-View.md)

#### 3. 冰山 (Iceberg) · 固态

- **功能定义：** 最终交付与资产管理
- **核心创新：** 资产不再是线性历史记录，而是一张**"灵感三角洲"拓扑图**
- **价值：** 不仅交付作品，更交付**蓄能潭**——独一无二的数字资产与创作基因
- **技术实现：** 见 [G3 冰山资产沉淀模块](../system-design/G3-冰山-资产沉淀模块设计.md)

#### 4. 湖泊 (Lake) · 工作区

承载创意活动的容器。预设湖泊：西湖、玄武湖、太湖、洱海、青海湖（承载文化意象）；用户可自定义命名。多湖之间通过"运河（引用）"或"合流（合并）"连通。详见 [跨湖连通设计](../system-design/D6-跨湖连通设计.md)。

## 第三篇 · 交互语言（How）

### 一、按钮词典

#### 生成与创造
| 常规 | 青萍命名 | 隐喻 |
| :--- | :--- | :--- |
| 新建 | **投石** | 向平静水面投入石子，打破平静 |
| 生成 | **降雨** | 灵感如雨水自上而下补给 |
| 导入 | **引流** | 把外面的水引到自己的池塘 |
| 随机 | **湍流** | 打破僵局，制造混乱 |

#### 编辑与修正
| 常规 | 青萍命名 | 隐喻 |
| :--- | :--- | :--- |
| 保存 | **固形** | 将流动的液体暂时凝固 |
| 复制 | **分蘖 (niè)** | 一个好的创意可以生出多个分株 |
| 粘贴 | **浸润** | 像水墨晕染，自然融合 |
| 撤销 | **退潮** | 如潮水退回，准备下一次冲击 |
| 删除 | **蒸发** | 温和的告别，并未消失，只是换了形态 |

#### 关联与图操作
| 常规 | 青萍命名 | 隐喻 |
| :--- | :--- | :--- |
| 新建关联 | **引流** | 在两个节点间开辟水道 |
| 查找路径 | **探源** | 寻找两个节点的最短逻辑链路 |
| 断开连接 | **截流** | 切断无效或错误的关联 |

#### 发布与分享
| 常规 | 青萍命名 | 隐喻 |
| :--- | :--- | :--- |
| 发布 | **决堤口** | 积蓄已久的能量，势不可挡地奔涌 |
| 导出 | **入海** | 创意离开系统，融入更广阔的海洋 |
| 分享 | **抛饵** | 抛出诱饵，期待引来更多的鱼 |
| 发送 | **漂流瓶** | 充满未知和浪漫的漂流 |

#### AI 与智能辅助
| 常规 | 青萍命名 | 隐喻 |
| :--- | :--- | :--- |
| 搜索 | **探源** | 寻找灵感的发源地 |
| 分析 | **透视** | 看清水下数据的本质 |
| 优化 | **疏浚** | 疏通堵塞，让创意流速更快 |
| 总结 | **沉淀** | 去伪存真，留下精华 |

### 二、空状态文案

| 触发场景 | 文案 |
| :--- | :--- |
| 无数据 / 404 | "**此处风平浪静，等待第一缕青萍。**" |
| AI 暂歇 | "**潮汐异常，AI 暂歇，请稍候。**" |
| 限流 | "**水流过急，请稍后再试。**" |
| 无权限 | "**此湖域未对外开放。**" |
| 误删恢复 | "**已从迷雾区取回 N 颗浮萍。**" |

### 三、配色（Color Scheme）

- **主色 — 青碧色** `#2E8B57`：生机、起点、自然
- **辅助色 — 日出橙** `#FF7F50`：爆发力、行动点（CTA 按钮统一）
- **背景色 — 深海蓝** `#003366`：深度、专业（适用于"冰山"等沉淀场景）

## 第四篇 · 写给使用者的信

致使用「青萍」的你：

请不要着急。
在这里，允许**退潮**，允许**回流**，允许**降雨**补给。
当能量的密度达到临界点时，请毫不犹豫地推开**决堤口**。

去吧，让世界看见你的波澜。

—— **「青萍 (Ripple)」团队** 敬上
