## 二、付费档位（Freemium + Pro + Team + Enterprise）

### 2.1 Free（免费版）
- 1 个湖泊
- 单湖 ≤ 200 节点
- 基础 AI（每月 100 次织网）
- 仅 Markdown 导出
- 单设备登录
- 社区支持

### 2.2 Pro · 个人版（¥38 / 月，¥298 / 年）
- 无限湖泊
- 单湖 ≤ 5000 节点
- AI 每月 2000 次织网 + 多 Agent Critic
- 全部导出格式（PDF / HTML / Notion）
- 多设备同步
- 移动端云霓
- 跨湖运河
- 邮件支持

### 2.3 Team · 团队版（¥98 / 人 / 月，年付 8 折）
- 包含 Pro 全部
- 多人实时协作（≤ 20 人 / 湖）
- 角色权限（OWNER / NAVIGATOR / PASSENGER / OBSERVER）
- 团队资产仓
- 审计日志可导出
- 工单支持

### 2.4 Enterprise · 企业版（议价，起 5 万 / 年）
- 包含 Team 全部
- 私有部署 / 数据驻留指定 Region
- SSO / SCIM
- 自定义 LLM 接入
- 合规报告（GDPR / 等保 2.0 三级）
- 7×24 SLA + 专属客户经理

## 三、AI 成本测算

### 3.1 单次调用成本（参考 GPT-4o + Claude 3.5 平均）

| 调用类型 | 平均输入 | 平均输出 | 单价 | 单次成本 |
| :--- | :---: | :---: | :--- | :---: |
| 双节点织网（Weaver） | 500 tok | 100 tok | $5/$15 per 1M | ~$0.004 |
| Critic 评审 | 800 tok | 50 tok | 同上 | ~$0.005 |
| 文档炸开（per chunk） | 1000 tok | 200 tok | 同上 | ~$0.008 |
| 单节点分蘖 | 300 tok | 150 tok | 同上 | ~$0.004 |

### 3.2 单用户月度成本估算

| 用户类型 | 月调用次数 | LLM 成本 | 存储 | CDN | **合计** |
| :--- | :---: | :---: | :---: | :---: | :---: |
| Free 用户 | 100 | ¥3 | ¥0.5 | ¥0.5 | **¥4** |
| Pro 用户 | 2000 | ¥60 | ¥3 | ¥2 | **¥65** |
| Team 用户 | 4000 | ¥120 | ¥10 | ¥5 | **¥135** |

### 3.3 毛利率测算（已落地成本压缩方案 3+4）

**优化策略（详见 [G5 §6.2/6.3](system-design/AI服务编排-织网工作流.md)）：**
- 多模型路由：Weaver 主用 Qwen2.5-72B 自建推理，仅复杂节点回退到 Claude Sonnet
- 本地 Embedding：BGE-large-zh 替代 OpenAI embedding
- 向量先过滤，只对候选节点对调 LLM

| 档位 | 售价 | 原成本 | 优化后成本 | 毛利率 |
| :--- | :---: | :---: | :---: | :---: |
| Free | ¥0 | ¥4 | **¥1.5** | -∞（获客） |
| Pro | ¥38 | ¥65 ❌ | **¥24** | **+37%** ✅ |
| Team | ¥98 | ¥135 ❌ | **¥50** | **+49%** ✅ |

### 3.4 ✅ 关键发现（已闭环）

原 Pro 定价亏损 71% 的问题已通过 G5 §6.2/6.3 的**多模型路由 + 本地推理**方案解决，毛利率转正。未来若 Token 单价下降 30%，还有进一步提价或补贴空间。

## 六、收支盈亏平衡测算

假设：
- 10 万 DAU
- Free → Pro 转化率 5% = 5000 付费
- 团队版 200 个团队 × 平均 8 人 = 1600 付费席位
- 企业 5 单 × ¥10 万 = ¥50 万

**预估年收入：**
- Pro：5000 × ¥298 = ¥1.49M
- Team：1600 × ¥98 × 12 = ¥1.88M
- Enterprise：¥0.5M
- **合计 ¥3.87M**

**结论：** ⚠️ 收入 ¥3.87M vs 成本 ¥17.88M = **缺口 ¥14M**。

**生存路径：**
1. 必须把 AI 成本砍 60%（方案 3+4）
2. 或必须把转化率提到 15%
3. 或必须把企业版做大到年 ¥5M+
4. 或接受 18-24 个月烧投资人钱

## 八、风险与应对

| 风险 | 应对 |
| :--- | :--- |

| 同类产品价格战 | 强化品牌（水文生态 = 不可复制 IP） |
