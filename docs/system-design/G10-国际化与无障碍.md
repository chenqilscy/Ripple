## 一、国际化 (i18n)

### 1.1 范围
- 默认语言：简体中文
- 一阶段支持：中、英
- 二阶段：繁体中文、日文

### 1.2 技术
- 前端：i18next + react-i18next
- 后端：仅做模板插值，文案下发给前端
- 资源文件：`locales/{lang}/common.json`

### 1.3 文化适配挑战

「青萍」品牌深植中文意境，国际化不是翻译，是 **重写**：

| 中文术语 | 直译（差） | 文化适配（好） |
| :--- | :--- | :--- |
| 投石 | Throw a stone | **Drop a Pebble** |
| 决堤口 | Break the dam | **Floodgate** |
| 蒸发 | Evaporate | **Evaporate**（保留） |
| 凝露 | Condense | **Crystallize** |
| 漂流瓶 | Drifting bottle | **Bottle in the Tide** |
| 朦胧版 | Misty version | **Skeleton View** |
| 舟子 / 渡客 | Boatman / Passenger | **Captain / Crew** |
| 风起于青萍之末 | (典故无法直译) | **A breeze begins with a single ripple.** |

> 原则：**保留水的隐喻，放弃古典出处**。

### 1.4 双向文本与排版
- 暂不支持 RTL
- 行高、间距按语言分别 token 化（中文 1.5 / 英文 1.6）

### 1.5 数据国际化
- 时间：ISO 8601 + 客户端按 timezone 渲染
- 数字：Intl.NumberFormat
- 用户内容：保留原语言，AI 在生成时可按用户偏好语言

## 二、无障碍 (a11y)

目标：**WCAG 2.1 AA**。

### 2.1 颜色对比
- 文本对比度 ≥ 4.5:1
- 暗色模式特别校验：节点白字在 qing-500 透明背景上对比 ≥ 4.5:1

### 2.2 键盘导航
- 全部交互可用键盘完成
- Delta View 节点支持方向键移动焦点
- `Enter` 进入编辑，`Esc` 退出，`Del` 触发蒸发（带二次确认）

### 2.3 屏幕阅读器
- 所有 SVG / WebGL 节点提供 `aria-label`（节点 content + 状态）
- 实时协作通知用 `aria-live="polite"` 区域
- 动效 / 涟漪：可关闭（`prefers-reduced-motion`）

### 2.4 焦点管理
- 模态框打开自动聚焦首项
- 关闭后归还到触发元素

### 2.5 动效降级
- 监听 `@media (prefers-reduced-motion: reduce)`
- 关闭：粒子流、涟漪扩散、决堤口动画
- 保留：状态颜色变化（保证信息传达）
